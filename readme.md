## Technologies Used

- Svelte
- TypeScript
- Vite
- Tailwind CSS

## Setup

To run this project locally:

1. Clone the repository
2. Install dependencies with `npm install`
3. Run the development server with `npm run dev`

## Building

To build the project for production:

```
npm run build
```
